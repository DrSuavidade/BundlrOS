# Build Resources

This folder contains build resources for the Electron app.

## Required Files

- `icon.ico` - Windows application icon (256x256 recommended)
- `icon.png` - PNG icon for macOS/Linux (512x512 recommended)

## Generating Icons

You can use tools like:

- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- [Online ICO converter](https://convertio.co/png-ico/)

Place your app icon as `icon.png` here, then generate the `.ico` version for Windows.
